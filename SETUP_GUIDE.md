# 防災 行動チェック - セットアップガイド

## プロジェクト概要

「防災 行動チェック」は、災害発生時に一般市民が「今やるべき行動」を即座に確認できるWebアプリケーションです。

### 主な特徴

- ✅ 地震・台風・大雨の3種類の災害に対応
- ✅ フェーズごとに整理された行動チェックリスト
- ✅ チェック状態の自動保存（localStorage使用）
- ✅ スマートフォン対応のレスポンシブデザイン
- ✅ シンプルで直感的なUI

## セットアップ手順

### 1. 必要な環境

- Node.js 20以上
- npm または yarn

### 2. プロジェクトディレクトリに移動

```bash
cd disaster-action-checklist
```

### 3. 依存パッケージのインストール

```bash
npm install
```

### 4. 開発サーバーの起動

```bash
npm run dev
```

ブラウザで http://localhost:3000 にアクセスしてアプリケーションを確認できます。

### 5. 本番ビルド（オプション）

```bash
npm run build
npm start
```

## アプリケーション構成

### 対応災害とフェーズ

#### 🏚️ 地震
- **平常時**: 非常用持ち出し袋の準備、家具の転倒防止確認、避難場所確認
- **発災直後**: 身の安全確保、火の元確認、正しい情報確認
- **避難時**: 非常用持ち出し袋を持つ、ブレーカーを落とす、避難場所へ移動

#### 🌀 台風
- **平常時**: ベランダの物を片付ける、雨戸・窓の確認
- **接近時**: 外出を控える、最新の気象情報確認
- **避難時**: 指示に従って避難、危険な場所に近づかない

#### 🌊 大雨
- **平常時**: ハザードマップ確認、側溝の詰まり確認
- **警報時**: 川や用水路に近づかない、避難情報確認
- **避難時**: 安全な場所へ移動、夜間は無理に移動しない

## 技術仕様

### 使用技術
- **Framework**: Next.js 16 (App Router)
- **言語**: TypeScript
- **スタイリング**: Tailwind CSS 4
- **状態管理**: React Hooks (useState, useEffect)
- **データ永続化**: localStorage

## 機能説明

### 1. トップページ（/）
- 3種類の災害カードを表示
- カードをクリックすると各災害の詳細ページへ遷移

### 2. 災害詳細ページ（/earthquake, /typhoon, /flood）
- フェーズタブで状況を切り替え
- 各フェーズの行動チェックリストを表示
- チェックボックスで完了状態を管理
- 完了状況を画面下部に表示

### 3. データ永続化
- チェック状態はlocalStorageに自動保存
- ページをリロードしても状態を保持
- 災害種別ごとに独立して管理

## カスタマイズ方法

### チェックリスト項目の追加・変更

`data/disasters.ts` ファイルを編集してチェックリスト項目を変更できます。

```typescript
{
  id: 'unique-id',        // 一意のID
  text: '行動項目の説明'  // 表示するテキスト
}
```

### スタイルのカスタマイズ

Tailwind CSSを使用しているため、各コンポーネントのクラス名を変更することで簡単にスタイルを調整できます。

## トラブルシューティング

### ポート3000が使用中の場合

```bash
npm run dev -- -p 3001
```

別のポートで起動できます。

### localStorageが動作しない場合

ブラウザのプライベートモードやCookie設定を確認してください。
